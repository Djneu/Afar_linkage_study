Files used in the study: 
"To Rotate or to Link? The Connection Between the Red Sea and the Gulf of Aden Rifts in Central Afar"
Ameha Muluneh, Sascha Brune, Carolina Pagli, Alessandro La Rosa, Derek Keir, Derek Neuharth, and Giacomo Corti

This consists of the prm file used to run the reference model, as well as an additional plugin to initially perturb the LAB.

The study was conducted using ASPECT version 2.4.0-pre (commit 84d40e7) and deal.II version 10.0.0-pre (commit 18a3861).
